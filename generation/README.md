# Welcome to discord gptbot _generation_
This directory contains the code for a simple python flask application that follows this flow:  
1. Flask Webapp starts up
2. Listens for requests on `/generate   
3. Parses an inboud request
4. Generates a sample of text based on the request
5a. Returns the generated text (200)
5b. If something goes wrong - a (500) is returned
